# Smart Luogu Dark

[Smart Luogu](https://userstyles.world/style/4990/smart-luogu)的深色主题分支版本。

## 安装

浏览器安装stylus插件，在管理样式页面中点击新建样式，将[smart-luogu-dark.css](./smart-luogu-dark.css)文件复制粘贴保存即可。
